# NEET Biotechnology Mock Test

A clean, dark-mode **NEET UG mock test website** focused on **Biotechnology and Its Applications (Chapter 10)**.

This project is designed to simulate a real exam-style environment with a timer, question navigator, answer tracking, negative marking, and detailed solutions after submission.

## Features

- 30 chapter-based NEET-style MCQs
- 45-minute countdown timer
- +4 / -1 marking scheme
- Skip and revisit functionality
- Question navigator sidebar
- Answer / skipped / remaining counters
- Instant score calculation
- Detailed solution review after submission
- Minimal dark UI for focused study

## Chapter Covered

**Biotechnology and Its Applications**  
Class 12 Biology · NEET UG

Topics included:
- Biotechnology applications
- GM crops
- Tissue culture
- Somatic hybridisation
- Bt cotton
- RNA interference (RNAi)
- Recombinant insulin
- Gene therapy
- PCR and ELISA
- Transgenic animals
- GEAC and Biopiracy
